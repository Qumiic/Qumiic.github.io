The Main Components of A Computer

CPU

Central Processing Unit.The primary component of a computer responsible for executing instructions and performing calculations. It is often referred to as the "brain" of the computer. The CPU interprets and carries out instructions from computer programs, handling tasks such as arithmetic operations, logic operations, and managing data movements within the computer's memory.

RAM

Random Access Memory. A temporary storage area that holds data and instructions that the CPU needs to access quickly. RAM allows the CPU to work with data rapidly, improving the computer's performance.

Storage Devices

Computers have various types of storage devices for long-term data retention. The most common ones are Hard Disk Drives (HDDs) and Solid State Drives (SSDs), which store the operating system, software, files, and data.

Motherboard

The main circuit board of the computer, serving as a platform for connecting and integrating all the essential components, such as the CPU, RAM, storage devices, and peripherals.

GPU

Graphics Processing Unit. A specialized processor designed to handle graphics-related tasks. It is crucial for rendering images, videos, and running graphics-intensive applications and games.

PSU

Power Supply Unit. Converts AC power from the electrical outlet into the DC power required by the computer's components.

Peripherals

Such as Input/Output devices like Keyboard, Mouse, Touchpads which are inputs and Monitors, Printers, Speakers which are outputs. Lastly, a Cooling system, typically composed of fans and heat sinks, ensures that the components remain at safe operating temperatures.

How to build a PC

Step By Step Guide


1. Prepare the Case
- Place the case on a flat surface. Open the side panel(s) to access the interior.

2. Install the Power Supply (PSU)
- Mount the PSU in the designated PSU compartment. Route the power cables to the backside of the case for later connection.

3. Install the CPU
- Carefully open the CPU socket on the motherboard. Align the CPU with the socket's alignment marker and gently lower it in. Lock the CPU in place.

4. Install the CPU Cooler
- Apply a small amount of thermal paste to the CPU's center. Attach the CPU cooler according to the cooler's instructions.

5. Install RAM
- Open the RAM slots on the motherboard. Align the RAM sticks with the notches on the slots and firmly press them in until they click.

6. Install Storage Drives
- Attach SSDs or HDDs in the designated drive bays using screws or trays provided.

7. Install the Motherboard
- Carefully align the motherboard's I/O shield with the case's rear I/O cutout. Place the motherboard on the standoffs within the case. Secure the motherboard using screws provided.

8. Connect Cables
- Connect power cables from the PSU to the motherboard (ATX power, CPU power, etc.). Connect SATA cables from storage drives to the motherboard.

9. Install the GPU
- Remove the appropriate expansion slot covers. Align the GPU with the PCIe slot and firmly press it in until it clicks. Secure the GPU using screws or brackets.

10. Cable Management
- Route cables neatly and use zip ties to secure them out of the way for better airflow.

Lastly, do a final check to ensure all connections and components are securey seated. Then connect the monitor, keyboard and mouse. And it's ready to boot up.
